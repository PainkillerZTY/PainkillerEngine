## Layer Dependencies

Each layer only depends on layers below it:

| Layer | Depends On |
|-------|-----------|
| Demo | Engine, Scene, Render |
| Scene/ECS | Core, Math, Render |
| Resource | Core, Render |
| Render | Core, Math |
| Engine | Core, Math, Render, Scene, Resource |
| Core/Math | GLM |

## Render Backend Interface

The `Renderer` class is a pure virtual interface. To add a new backend:

1. Create `D3D12Renderer.h/.cpp` that inherits from `Renderer`
2. Implement all pure virtual methods
3. Add to `Renderer::Create()` factory
4. Done ? no changes needed in Scene, Engine, or Demo
